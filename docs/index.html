Orchestrator Kit for Agentic Reasoning

YAML-based workflow orchestration. Redis-backed memory. Traceable execution.

Key Features

  • Agent-based architecture: Build modular workflows with different agent types.
  • YAML-defined logic: Design workflows using YAML configuration files.
  • Redis streaming memory: Agent interactions are logged and stored in Redis.
  • Traceable execution: Full execution history — every step in the pipeline is logged.
  • LLM integration: Integrate OpenAI, Ollama, and more as agents in workflows.
